"Flexbox: O Guia Simples para Layouts Flexíveis"

O que é Flexbox?

Flexbox é um módulo de layout do CSS que facilita a criação de layouts flexíveis e responsivos. Com o Flexbox, você pode organizar seus elementos na página de maneira eficiente, mesmo quando as dimensões dos elementos são dinâmicas ou desconhecidas

Como Funciona o Flexbox?

Flexbox organiza elementos em um contêiner principal chamado 'container flex'. Dentro desse contêiner, os elementos filhos são chamados de 'itens flex'. O grande diferencial do Flexbox é que ele permite distribuir espaço entre os itens e alinhar esses itens de maneira previsível, mesmo que o tamanho de um ou mais itens seja desconhecido ou dinâmico.

Propriedades Principais do Container Flex

  • display: flex; Ativa o Flexbox no contêiner. Todos os filhos diretos deste container se tornam itens flexíveis.
  • flex-direction Define a direção dos itens flexíveis. Pode ser row (linha), column (coluna), entre outros.
  • justify-content Controla o alinhamento dos itens ao longo do eixo principal. Permite centralizar, distribuir espaço igualmente, etc.

Propriedades dos Itens Flexíveis

  • flex-grow Permite que um item flex cresça e ocupe o espaço disponível no contêiner. Por exemplo, flex-grow: 1; faz com que o item cresça em proporção ao espaço disponível.
  • flex-shrink Define como um item deve encolher se o espaço no contêiner for insuficiente.
  • align-self Permite que um único item flex se alinhe diferentemente dos outros ao longo do eixo transversal.
